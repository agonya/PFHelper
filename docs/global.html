GenerateBlankArray(n)

Generate a blank array in which the value of the elements of the array are equal to their index numbers.
This way, you can just use "for (let i of generateBlankArray(n))" instead of using the "i" syntax.
This is for simplifying for loops and must only be used for the most basic cases.
Parameters:
Name Type Description
n number Number of elements in the array. 0: generates a blank array with length 0

GetColor(colorString, subCode, hex)

Converts the color from color palette to code usable Phaser Color Type.
Colors:
red, pink, purple, deeppurple, indigo, blue, lightblue, cyan, teal, green,
lightgreen, lime, yellow, amber, orange, deeporange, brown, grey, bluegrey
Subcodes:
50, 100, 200, ... 900, (a100, a200, a400, a700)
Parameters:
Name Type Description
colorString string A string representing the main name of the color e.g. "red"
subCode string A subcode to the main color which darkens it as it is increased.
hex boolean true: returns with "#" prefix false: returns a number
